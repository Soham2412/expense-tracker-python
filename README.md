# Expense Tracker (Python)

A simple Python project that tracks daily expenses using a CSV file.

## Features
- Add new expenses
- Read and analyze stored expense data
- Calculate total spending
- Categorize expenses

## Files
- `expense_tracker.py` → main program
- `expenses.csv` → sample expense data (for demo/testing)

## Notes
The CSV file contains sample data only and does not represent real personal financial records.
